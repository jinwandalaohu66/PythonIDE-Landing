## ✨ 核心功能 / Core Features

### 🐍 多语言运行
| 功能 | 描述 |
|------|------|
| **Python 3.13** | 本地运行、标准库、交互式 I/O、NumPy / Pillow / requests 等预装 |
| **JavaScript** | JavaScriptCore 执行 .js，alert/confirm/prompt、fetch、localStorage、剪贴板等桥接 |
| **HTML 预览** | WKWebView 加载 HTML+CSS+JS，相对路径、alert/console 桥接 |

### ✏️ 专业编辑
| 功能 | 描述 |
|------|------|
| **语法高亮** | Python、JavaScript、HTML、CSS、JSON、Markdown |
| **行号 & 缩进** | 等宽字体、自动缩进、可调字体大小 |
| **搜索** | 查找、替换、跳转行 |

### 📂 文件与工作流
| 功能 | 描述 |
|------|------|
| **多层级文件夹** | 创建、重命名、移动、删除、回收站（7 天保留） |
| **分屏模式** | 编辑器与控制台同屏，边写边看输出 |
| **Rich 输出** | 彩色输出、图表可视化、ANSI 支持 |

### 🛠️ 开发者工具箱
| 功能 | 描述 |
|------|------|
| **JSON 格式化** | 美化、压缩、验证 |
| **编解码** | URL、Unicode、Base64、MD5/SHA256 |
| **API 调试** | HTTP 请求、响应预览 |
| **二维码** | 生成、识别、Data URL 转换 |
| **正则测试** | 实时匹配、分组预览 |
| **时间戳** | 毫秒/秒互转、格式化 |

### 🤖 AI 助手（需自配 API）
- 自然语言改代码
- 审查、注释、优化、修错
- 对话式聊天，支持多轮重试

### 🔒 隐私与体验
- **Face ID / Touch ID** 应用锁定  
- **Photos 模块**：相册、相机、Pillow 图像处理  
- **媒体预览**：图片、视频、PDF  
- **灵动岛**：脚本运行状态实时显示
